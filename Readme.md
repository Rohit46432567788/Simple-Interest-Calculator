# Simple Interest Calculator

## 📌 Project Description
This project is a simple web-based application that calculates Simple Interest based on user input.

## 🧮 Formula Used
Simple Interest = (Principal × Rate × Time) / 100

## 🚀 Features
- User inputs principal amount, rate of interest, and time
- Calculates simple interest instantly
- Displays the result clearly
- Simple and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📷 Output
The application takes user input and displays the calculated simple interest.

## 📂 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter values and calculate interest

## 📌 Example
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = 100
